# Journey for App
### YuLife App Design System

A living component library for the YuLife mobile app. All components are built with **React Native Web** so they render faithfully in the browser — no device or simulator required.

Each component is linked back to Figma via **Figma Code Connect**, so the code snippets you see in Figma Dev Mode always match what's here.
