# Journey for App
### YuLife App Design System

A living component library for the YuLife mobile app. All components are built with **React Native Web** so they render faithfully in the browser — no device or simulator required.

Each component is linked back to Figma via **Figma Code Connect**, so the code snippets you see in Figma Dev Mode always match what's here.

## Changelog

### v1.1 — 24 April 2026
- **InlineBanner** toast/banner component with icon, message, optional CTA, and auto-dismiss support
- **Popover** floating tooltip / onboarding bubble with four-direction beak, enter/exit animation, outside-tap and close-button dismiss, and optional CTA action
- **UnorderedList** component with nested list support using design system text and spacing tokens
- **Animation** token file and story added (`duration` and `easing` scales)
- **Button** gains four status colour variants — `Success`, `Warning`, `Error`, `Info` — for use as action buttons inside InlineBanner
- **ModalTemplate** `hero` prop made optional (modal can now render without a hero); animation durations switched from hardcoded values to `animation` tokens
- **Color tokens** — `palette.transBlack8` (black at 8% opacity) added
- **Colour Icons** gallery now displays in a multicolumn grid and icons are sorted alphabetically
- **Line Icons** gallery now displays in a multicolumn grid
- **Storybook** docs story default height reduced from 852 px to 300 px
- **Radii** Usage Examples story removed; Scale and Token Table stories remain

### v1.0 — 24 April 2026
- **Storybook** manager themed with design tokens — purple800 accent, sidebar logo, favicon, and white canvas background
- **Welcome page** section grid added with live story thumbnails for each category
- OG meta tags added for Slack / social link previews

### v0.9 — 24 April 2026
- **YugiStatus colour icons** (`YugiStatusErrorColourIcon`, `YugiStatusWarningColourIcon`, `YugiStatusSuccessColourIcon`, `YugiStatusInfoColourIcon`) exported from the main `@/icons` barrel; replaced broken CSS-mask SVG compositing with PNG assets (96 × 96 source) from Figma
- **ColourIconSize** scale tightened to 8 px increments from 16 to 48 (16, 24, 32, 40, 48) — removes non-standard steps 20, 28, 36, and 64

### v0.8 — 24 April 2026
- **Elevation** token category added (`src/tokens/elevation.ts`) with `none` and `toast` levels; shadow colour mapped to new `palette.transBlack8` (black at 8% opacity)
- **InlineBanner** toast shadow updated to use `elevation.toast` token (sourced from Figma node 11048:4475)
- **Welcome page** section grid added above the Changelog — five cards (Foundations → Pages) each with a live story thumbnail and description

### v0.7 — 22 April 2026
- **Checkbox** component with checked, disabled, and inactive states — uses the design system `CheckIcon`
- **Radio** component (bare 24×24 radio knob) and **RadioGroup** component grouped under a shared `Radio/` parent folder
- **Toggle** component added
- Storybook upgraded to v10; all stories migrated to `@storybook/react-vite`

### v0.6 — 22 April 2026
